##  Control System Security
自动驾驶汽车中的各种机械部件和数字设备均由 ECU 控制。车辆中不同ECU之间的通信通过数字总线进行。黑客可以通过车载诊断 (OBD) 端口侵入 CAN 总线网络。
关系不大，暂不收集。
